Diferencia entre display: flex y display: inline-flex

Los contenedores con display: flex ocupan todo el ancho disponible (como elementos de bloque), mientras que los contenedores con display: inline-flex ocupan solo el espacio necesario para su contenido (como elementos en línea).

display: flex

Este contenedor se comporta como un elemento de bloque (ocupa todo el ancho):

1
2
3

Este texto está después del contenedor flex.

display: inline-flex

Este contenedor se comporta como un elemento en línea (solo ocupa el espacio necesario):

1
2
3
Este texto está al lado del contenedor inline-flex.

Ejemplo con texto alrededor

Este es un texto normal que fluye alrededor de los contenedores.

F
L
E
X
Como puedes ver, el contenedor flex crea un salto de línea antes y después, ya que se comporta como un elemento de bloque.

Este es otro texto normal que fluye alrededor de los contenedores. I-F L E X Como puedes ver, el contenedor inline-flex se integra con el texto, ya que se comporta como un elemento en línea.
