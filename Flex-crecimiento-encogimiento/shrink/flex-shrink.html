Comprendiendo flex-shrink

La propiedad flex-shrink define la capacidad de un elemento flex para encogerse si es necesario. Determina cuánto se reducirá un elemento en relación a los demás cuando no hay suficiente espacio en el contenedor.

Ejemplo 1: Todos los elementos con flex-shrink: 1 (valor predeterminado)

Todos los elementos se encogerán por igual cuando no haya suficiente espacio:

1
flex-shrink: 1
2
flex-shrink: 1
3
flex-shrink: 1

Ejemplo 2: Un elemento con flex-shrink: 0

El segundo elemento no se encoge, los otros sí:

1
flex-shrink: 1
2
flex-shrink: 0
3
flex-shrink: 1

Ejemplo 3: Diferentes valores de flex-shrink

El elemento 1 se encoge más que el elemento 3, y el elemento 2 no se encoge:

1
flex-shrink: 2
2
flex-shrink: 0
3
flex-shrink: 1
